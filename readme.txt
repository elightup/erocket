Supercharge your WordPress themes with extra functionalities and modules.

== Description ==

Extend your theme functionality with eRocket with various modules like custom widgets and social media sharing buttons.

### Features

- Contact info widget
- Recent posts widget
- Social sharing buttons

== Installation ==

1. Go to Plugins > Add New
1. Search for "erocket"
1. Install and activate the plugin
